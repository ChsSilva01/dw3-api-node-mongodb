### MERN STACK

o que é? uma coleção de tecnologias para a construção de uma aplicação em todas as camadas da programação.



M - mongoBD: Banco de dados noSQL (documentos)

E - EXPRESS: FrameWork JS (construir API)

R - REACT: Biblioteca JS (construir UIs)

N - NodeJS: Ambiente de execução JS

# 

#### **MongoDB**

##### **> O que é?**

É um banco de dados NoSQL orientado a documentos armazena em formato semelhante a JSON



##### **> Por que usar?**

É altamente escalável e flexível além de armazenar grandes quantidades de dados



##### **> Vantagens**

Fácil de se utilizar e tem compatibilidades com várias aplicações diferentes

# 

#### **Express**

##### **> O que é?**

express é um framework minimalista para nodeJS. Utiliza para montar API's e Sites robustos



##### **> Por que usar?** 

Desenvolvimento simples com uma boa manipulação de rotas



##### **> Vantagens**

Fácil de se utilizar é contém os middlewares além de outras vantagens

# 

### **React**

// Biblioteca

// Ele não é lincado é sim atualizado sendo que seu código e sempre reutilizado



##### **> O que é?**

Biblioteca javaScript utilizada para construir interfaces de usuários interativas e reutilizáveis.



##### **> Por que usar?**

Fácil de programação e criação de componentes que podem ser reutilizados.



##### **> Vantagens**

Eficiente através do virtual DOM, comunidade ativa com boas ferramentas

# 

### **Node.JS**

##### **> O que é?**

ambiente de execução javaScript no servidor, baseado no motor V8



##### **> Por que usar?**

Aplicações com alta performace I/O não bloqueante



##### **> Vantagens**

Grandes quantidade de módulos através do NPM é adequação para tempo real
